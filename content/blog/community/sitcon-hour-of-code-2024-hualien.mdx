![SITCON Hour of Code 2024 於國立東華大學](/images/blog/sitcon-hoc-2024/sitcon-hoc-2024.jpg)

SITCON Hour of Code 2024 花蓮場順利結束啦！感謝各位奔赴花蓮國立東華大學 (NDHU)，以及我們的 Host，YC Huang 承辦這場有趣的活動。

## 計畫趕不上變化

在事前會議上，我們還在討論如何控制難易程度。

結果現場看到各位小朋友天賦異稟，其實都蠻厲害的，開場不到 15 分鐘就完成最後的挑戰啦！只好給出更依賴邏輯架構與思考的 Celebrating 50 years of Kids Coding 給成員們挑戰。

![學員專注解題](/images/blog/sitcon-hoc-2024/sitcon-hoc-2024-2.jpg)

其實蠻有趣的。其中有一群小孩表示，在學校中有學過類似的。我覺得從小培養這些邏輯思考，是對未來來說一件不可或缺的事情。

![親子一起學習](/images/blog/sitcon-hoc-2024/sitcon-hoc-2024-3.jpg)

## 不只是孩子在學習

也有家長陪同參加，亦有家長本身是這次學員，大家專注沈浸在這樣的學習氛圍中。

看到家長和孩子一起討論程式邏輯，互相切磋解題思路，這樣的畫面特別溫馨。學習真的不分年齡，有時候親子一起摸索，反而能激發出更多火花。

![志工協助教學](/images/blog/sitcon-hoc-2024/sitcon-hoc-2024-4.jpg)

## 志工們的熱情

非常感謝從外縣市來的志工們，舟車勞頓特地來參加這次的活動。

活動能夠圓滿成功要特別感謝每一位志工的付出和努力。你們的熱情和專業不僅讓活動順利進行，更為花蓮的程式教育注入了新的活力。

![活動大合照](/images/blog/sitcon-hoc-2024/sitcon-hoc-2024-5.jpg)

花蓮離台灣的科技重鎮有點距離，要志工為了一天的工作坊跑到這裡真的不容易。但大家還是來了，因為我們相信程式教育不應該只是都會區孩子的特權。

![課程進行中](/images/blog/sitcon-hoc-2024/sitcon-hoc-2024-6.jpg)

## 期待下次相聚

期待下次能再相聚，一起為推廣程式教育繼續努力！

如果你有興趣參與 SITCON 志工活動，或是想把 Hour of Code 帶到你的地區，請留意 [SITCON 的公告](https://sitcon.org)。
